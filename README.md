# Exchange Board

Exchange management platform built with Laravel, API integrations and bot-related modules.

## About the project

Exchange Board is a Laravel-based project focused on exchange/payment workflows, support logic, API endpoints and bot integrations.

This project was created as a practice and portfolio project to work with a more business-oriented application structure:
users, balances, support flows, exchange operations, API routes and service-level logic.

## Tech Stack

### Backend
- Laravel 10
- PHP 8.1+
- Laravel Sanctum
- beyondcode/laravel-websockets
- Pusher
- Spatie Laravel Permission
- Guzzle

### Frontend
- Vue 3
- Bootstrap 5
- Vite
- Axios
- Laravel Echo
- Chart.js

## Main Features

- Exchange/payment related API
- Support-related routes and events
- Bot / Telegram-style integration modules
- Binance-related endpoints
- User and personal area logic
- Balance / top-up / withdrawal flows
- Broadcast channels
- Jobs, services and event-driven structure

## Architecture Highlights

The project includes:
- API routes
- Broadcast channel configuration
- Console-related structure
- Jobs
- Services
- Events
- Models

This makes it closer to a real business backend than to a simple CRUD demo.
